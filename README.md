# Kodo

Interfaz web para **generar y enviar códigos secretos**. Pequeño proyecto experimental enfocado en cifrado básico desde el navegador.

## Qué hace

- Codifica un mensaje en un código personalizado
- Genera el resultado listo para copiar y compartir
- Decodifica el código de vuelta al mensaje original

## Stack

- HTML5 · CSS3 · JavaScript (vanilla)

## Uso local

```bash
git clone https://github.com/felixBD04/kodo.git
cd kodo
# Abre index.html en tu navegador
```

## Por qué este proyecto

Práctica de:
- Lógica de cifrado básico implementada desde cero
- Manipulación del DOM para feedback en tiempo real
- Diseño de UIs sencillas pero funcionales sin librerías externas
